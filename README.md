## ✨ 项目简介

这是一个**交互式大模型进化时间轴**，覆盖 2018-2025 年全球主要 AI 公司的大语言模型发展脉络。

### 特性

- 🎯 **48 个模型** — 覆盖 13 家公司/团队的代表性大模型
- 🌍 **全球视野** — 🇺🇸 美国 / 🇨🇳 中国 / 🇪🇺 欧洲，国内外模型对比
- 🔍 **交互筛选** — 按地区、按公司自由过滤
- 💡 **悬停详情** — 发布时间、参数规模、能力标签一目了然
- 🏆 **旗舰标记** — 粉色圆点标识各公司旗舰模型
- 📱 **响应式设计** — 桌面端、移动端均可浏览
- 🚀 **零依赖** — 单个 HTML 文件，浏览器直开，无需任何构建工具

## 🚀 快速开始

### 方式一：直接打开

```bash
# 克隆仓库
git clone https://github.com/LLing486/llm-evolution.git
cd llm-evolution

# 浏览器打开
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

### 方式二：GitHub Pages

访问 **https://lling486.github.io/llm-evolution/** 即可在线体验。

## 🏗️ 技术栈

- **HTML5** + **CSS3** + **Vanilla JavaScript**
- 暗色主题，CSS Grid/Flexbox 布局
- 无任何外部依赖（Google Fonts 为可选增强）
