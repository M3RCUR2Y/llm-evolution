# LLM Evolution Timeline — Round 2 改进方案

## 目标：从"能看"升级到"好用"

### 1. 视图切换（时间线 ↔ 表格）
- 顶部加一个切换按钮：🕐 时间线视图 / 📋 表格视图
- 表格视图：可排序的表格，列包括：模型名、公司、发布时间、参数、标签
- 点击列头可排序
- 保持筛选条件联动

### 2. 模型对比功能
- 每个节点左边加一个小复选框（hover 时显示）
- 最多选 3 个模型
- 选中后底部弹出对比面板，显示：名称、公司、发布时间、参数、描述、标签
- 对比面板可关闭

### 3. 丰富筛选维度
- 在筛选栏加"类型"分组：开源 / 闭源 / 全部
- 在筛选栏加"模态"分组：纯文本 / 多模态 / 全部
- 需要在 data/models.json 中为每个模型添加可选字段：
  - "open_source": true/false/null
  - "modalities": ["text"] 或 ["text", "image", "audio"]

### 4. 改进搜索体验
- 搜索框右侧加结果计数（如 "3 / 48"）
- 搜索时自动展开匹配模型所在的公司行（如果之前被筛选隐藏）

### 5. 键盘快捷键
- `/` 聚焦搜索框
- `Esc` 清除搜索/关闭对比面板
- `←` `→` 水平滚动时间线

## 技术约束
- 仍然是 index.html + 外部 JSON
- 不引入框架
- 保持暗色主题
- 更新 data/models.json 添加新字段（open_source, modalities）
